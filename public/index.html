Rails OQGraph demo

What's this about?

This is a demonstration app showing the use of the acts_as_oqgraph gem for Ruby on Rails

The source code shows how to use the gem in your own applications.

The demo shows how fast the OQGraph engine is at dealing with graph traversal problems in a MySQL database.

The Demos

There are currently 2 demos: mazes and friends.

The mazes demo shows the use of OQGraph for maze solving, and the friends demo shows its use for the social networking problem of getting friends of friends.
